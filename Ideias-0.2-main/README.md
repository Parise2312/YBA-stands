#Anotações Importantes#

passo a passo como enviar arquivos em githubs 
1. rm -rf .git
2. git init
3. git add .
4. git commit -m "Sandia" (*exemplos*)
5. git branch -M main
6. git remote add origin https://github.com/Parise2312/Meu-Reposit-rio
7. git push -u origin main
